# Fake Stream Chat Overlay

게임 유튜버/스트리머 영상 촬영용 투명 채팅 오버레이입니다.

## 실행

```powershell
npm install
npm start
```

또는 `run.bat`을 더블클릭하세요.

## 현재 기능

- 투명 채팅 오버레이 창
- 컨트롤 패널에서 실시간 설정 변경
- 채팅 인원, 채팅 간격, 애니메이션 속도 조절
- YouTube, Chzzk, SOOP 플랫폼 뱃지 노출 설정
- 아이디 랜덤 색상
- 채팅창 투명도 조절
- 항상 위 표시
- 시청자 수 랜덤 워크 표시
- 감탄사, 질문, 게임 반응, 화면 키워드 기반 자동 채팅
- 오버레이 클릭 통과 모드
